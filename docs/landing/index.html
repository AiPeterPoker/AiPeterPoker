Features

Math-driven decisions.
Peter-level confidence.

Every component is designed for transparency and entertainment — perfect for streaming on Twitch or YouTube.

MC

Monte Carlo engine

10,000+ simulated runouts per hand. Full hand evaluation from royal flush to high card. Calculates equity, outs, and hand strength in real-time.

GT

GTO strategy

Pre-computed Game Theory Optimal tables for Casino Hold'em. Pre-flop categorization and post-flop strength tiers. Configurable strictness from pure GTO to gut-driven.

AI

AI vision

Screenshots the browser and uses Claude or GPT-4o to read the table. Detects cards, pot size, buttons, and game phase automatically.

TV

Twitch integration

Chat votes on decisions in real-time. Viewers type !fold, !call, or !raise. Results show in the overlay. Democracy mode for maximum entertainment.

PG

Peter's personality

Three personality modes: Overconfident, Cautious, and Degenerate. Each changes the console commentary, risk tolerance, and quip style. The math stays the same.

K$

Kelly criterion sizing

Automatic bet sizing based on edge and bankroll. Fractional Kelly for safety. Peter might be reckless, but the math isn't.

Get started

Deploy your own Peter
in 3 minutes.

Clone the repo, add your API key, and start. Peter runs locally on your machine — your keys never leave your computer.

# Clone the repo git clone https://github.com/ai-in-peter/ai-in-peter.git cd ai-in-peter # Install everything bash scripts/setup.sh # Add your API key cp .env.example .env nano .env # Add ANTHROPIC_API_KEY or OPENAI_API_KEY # Launch Peter npm start 🃏 AI-IN Peter — Starting... ✅ Backend running on ws://localhost:8765 ✅ Electron overlay loaded ✅ Peter is ready.
